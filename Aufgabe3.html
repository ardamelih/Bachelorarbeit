Wie kann ich die Sichtbarkeitsstufen für Anzeigen einstellen?

Sie wollen die Sichtbarkeitsstufen für Anzeigen einstellen? Dann sind Sie hier genau richtig!
Sie haben eine Idee für eine Anzeige, möchten die Anzeige aber zunächst einmal nur als Entwurf speichern oder im kleinen Kreis besprechen? Dann sollten Sie die Sichtbarkeit der Anzeige an Ihre Bedürfnisse anpassen.
Sie können die folgenden Sichtbarkeitsstufen für Anzeigen wählen:

  • Nur für Sie: Die Anzeige bleibt ein Entwurf, den nur Sie sehen können. So können Sie die Anzeige "reifen" lassen und erst dann veröffentlichen, wenn Sie damit glücklich sind.
    Diese Sichtbarkeitsstufe kann auch noch einen zweiten Zweck erfüllen: Indem Sie eine Anzeige zunächst nur für sich selbst erstellen, können Sie – nachdem die Anzeige gespeichert ist – von der Intelligenz von meinDorf55+ profitieren. Denn das System sucht auch für private Anzeigen andere Anzeigen, die dazu passen können. Stellen Sie sich vor, Sie sind auf der Suche nach einer Skatrunde. Sie könnten den Bereich Handeln natürlich auch mit der Suchfunktion direkt durchsuchen. Aber können Sie sich sicher sein, dass dann auch eine passende Skatrunde gefunden wird? Indem Sie Ihre Wünsche an eine Skatrunde formulieren – zum Beispiel, dass beim Spielen auch zünftig gegessen wird – geben Sie der meinDorf55+-App eine Chance, gut passende Skatrunden für Sie zu finden. Schauen Sie deshalb bei privaten Anzeigen auch immer wieder einmal in den Bereich "Ähnliche Inhalte".
  • Nur für Verantwortliche und Teilnehmende des Projekts: Vielleicht fragen Sie sich, was Anzeigen mit Projekten zu tun haben? Nun, Mitarbeitende in einem Projekt können vom Projekt aus eine Anzeige schreiben. Diese kann dann von allen Mitarbeitenden des Projekts bearbeitet werden. Ist sie zudem nur für Menschen mit Bezug zum Projekt sichtbar, so richtet sich die Anfrage ganz konkret nur an die Teilnehmenden und die Verantwortlichen im Projekt. Wenn Sie also für Ihr nächstes Treffen im Projekt nach Menschen suchen, die einen Kuchen backen können, so könnten Sie eine Anzeige aufgeben und diese auf den Personenkreis der Projektteinehmenden und der Projektverantwortlichen beschränken.
  • Für alle Mitglieder von meinDorf55+: Der Regelfall wird sein, dass Anzeigen für alle Mitglieder von meinDorf55+ sichtbar sind. Das Anliegen der Anzeige soll ja möglichst viele Menschen begeistern und zu Reaktionen anregen. Beachten Sie, dass Menschen in meinDorf55+ zunächst ihren Wohnort bestätigen lassen müssen, bevor sie Anzeigen sehen können.
